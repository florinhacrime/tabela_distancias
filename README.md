### Descrição 

O projeto consiste na composição de uma tabela que conta com capitais 
brasileiras e a distancia entre essas capitais.As capitais nesse projeto são:

* Cidade 
* MANAUS
* NATAL 
* PALMAS
* PORTO ALEGRE
* PORTO VELHO
* RECIFE
* RIO BRANCO
